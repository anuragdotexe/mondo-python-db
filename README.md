# MongoDB Task Manager (Python) 🛠️

A lightweight Command Line Interface (CLI) application built with **Python** and **MongoDB Atlas**. This project demonstrates full CRUD (Create, Read, Update, Delete) operations in a NoSQL environment.

## 🚀 Features
- **Cloud Integration**: Connects to MongoDB Atlas for persistent storage.
- **Task Management**: Create, view, update, and delete tasks in real-time.
- **Secure Credentials**: Uses environment variables to protect database passwords.
- **Priority Tracking**: Categorize tasks by Low, High, or Critical priority.

## 🛠️ Tech Stack
- **Language:** Python 3.10+
- **Database:** MongoDB Atlas (NoSQL)
- **Library:** PyMongo (Official MongoDB Driver)
- **Environment Management:** Python-Dotenv
